# Personal website

My personal website where i publish my blogs and other stuffs

## Run Locally

Clone the project

```bash
  git clone https://github.com/devpenzil/devpenzil.dev.git
```

Go to the project directory

```bash
  cd devpenzil.dev
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
